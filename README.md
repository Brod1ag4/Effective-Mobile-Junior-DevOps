Как собрать и проверить проект
1 - Перейдите в каталог с проектом;
2 - Введите команду, чтобы собрать контейнер: "docker compose up -d";
3 - На хосте выполните команду: "curl http://localhost". Результат должен быть - "Hello from Effective Mobile!";

Как работает схема nginx → backend

Пользователь отправляет запрос на http://localhost
Nginx получает запрос и перенаправляет его на backend:8080
Backend (сервер) обрабатывает запрос и возвращает текст "Hello from Effective Mobile!"
Nginx передает ответ обратно пользователю
